# Missed Connections
##### Highlights any tracks that are not connected to another track or station with a red X

You no longer need to hunt for breaks in your tracks when you get the dreaded "No valid path found between station tracks" error.

By default, the mod starts disabled.  You have two ways to use this mod:
1. In the top-right UI bar, click the associated button. It looks like a cable with connection plugs on each end. It will toggle on and change color. Click it again to toggle it off.
2. Hold the ~ key to momentarily toggle the mod.  If it's currently disabled, it will enable the map layer, and if it's currently enabled, it will disable the map layer.

Happy building.
